## Backend Installation

```bash

# Go to 'app' Director
$ npm install

########### Running the app ###################

# development
$ npm run start

# watch mode
$ npm run start:dev

# Application is running on port 5000

```

## Frontend Installation

```bash

# Go to 'web' Director

$ npm install
$ npm install -f

########### Running the app ###################

# watch mode
$ npm run dev

# Application is running on port 3000
# http://localhost:3000

```
